1.1- ¿Para qué sirve la etiqueta ul? Dar un ejemplo en donde lo utilizarías.

ul de "unordered list" crea una lista no ordenada. Esto quiere decir que no tien ni un orden numerico o alfabeto, lo podria utilizar para una lista nombres.

1.2- ¿Para qué sirve la etiqueta ol? Dar un ejemplo en donde lo utilizarías.

Esta etiqueta sirve para definir listas ordenadas de manera numerica o alfabeta. ol es de "Ordered List".Por ejemplo este lo utilizaria en una lista de ingredientes

1.3- ¿Para qué sirve la etiqueta dl? Dar un ejemplo en donde lo utilizarías.

La etiqueta dl representa una "Lista descriptiva". Es decir que esta elemento encierra un lista de terminos y descripciones, un ejemplo para su uso podria ser de uso personal.

Ejercicio 2: Enlaces

2.1- ¿qué es una url absoluta?

Estas proporcionan la URL completa, incluido el protocolo (http: // o https: //), el nombre de dominio y la ruta como necesario. Debe usar una URL absoluta cuando apunta a un documento en la web es decir, no en su propio servidor.

2.2- ¿qué es una url relativa?

Las URL relativas describen la ruta a un archivo relativo al actual documento. Las URL relativas se pueden usar cuando se vincula a otro documento en su propio sitio. No requiere el protocolo o nombre de dominio, solo el nombre de ruta.
